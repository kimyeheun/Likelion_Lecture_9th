### 터미널이란?

- CLI를 GUI환경에서 사용할 수 있게 해주는 것.



#### CLI = Command Line Interface

-> 그래픽 없이 글로만 명령어로 컴퓨터를 조작하는 인터페이스

#### GUI = Graphic User Interface

-> 그래픽으로 일반 사용자들고 쉽게 사용할 수 있는 인터페이스
